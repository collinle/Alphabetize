This challenge uses encryption keys to map each letter of a word to a different letter of the alphabet. For example, if our key was 'czylkhwedjvuinxfbspatmgroq', the letter 'a' would be mapped to 'c', 'b' would be mapped to 'z', and so on. An encryption key's score is the number of words from a list of 7260 six-letter words that the key maps to an alphabetized string. The goal of the challenge is to find the key with the highest score. Click the button to start the program.

The program initially produces a random reordering of the 26 letters of the alphabet as the initial encryption key. Pressing 'Next step' instructs the program to make small rearrangements of the current key until one with a better score is found. The table displays the current key, the score using the current key, and the number of letter switches between the previous key and the current one. Keep pressing 'Next step' to generate better keys!
